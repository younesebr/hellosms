# HelloSms - ارسال خودکار پیامک پس از رد یا از دست رفتن تماس

برنامه اندروید حرفه‌ای نوشته شده با پایتون که به صورت خودکار پس از رد یا از دست رفتن تماس، پیامک را برای تماس‌گیرنده ارسال می‌کند.

## ویژگی‌ها

- ✅ **مانیتورینگ خودکار تماس‌ها**: تشخیص رد یا از دست رفتن تماس به صورت خودکار
- ✅ **ارسال خودکار پیامک**: ارسال فوری پیامک پس از قطع تماس
- ✅ **قابل تنظیم**: متن پیامک قابل تغییر و ذخیره در داخل برنامه
- ✅ **سازگار با Android 11+**: طراحی شده برای اندروید 11 و بالاتر
- ✅ **رابط کاربری ساده**: رابط کاربری فارسی و کاربرپسند
- ✅ **فعال/غیرفعال کردن**: امکان خاموش/روشن کردن سرویس

## نیازمندی‌ها

- Python 3.11+
- Kivy 2.1.0
- Buildozer (برای ساخت APK)
- Android SDK و NDK
- Java JDK

## نصب و راه‌اندازی

### 1. نصب وابستگی‌ها

```bash
pip install -r requirements.txt
```

### 2. تست در دسکتاپ (اختیاری)

```bash
python main.py
```

### 3. ساخت APK برای اندروید

**روش ساده (توصیه می‌شود): استفاده از GitHub Actions**

1. یک حساب GitHub بسازید (رایگان)
2. یک repository جدید بسازید (حتماً PUBLIC باشد)
3. همه فایل‌های HelloSms را آپلود کنید
4. به تب "Actions" بروید
5. "Build APK with Buildozer" را اجرا کنید
6. 15-30 دقیقه صبر کنید
7. APK را از بخش "Artifacts" دانلود کنید

**برای راهنمای کامل:** فایل `HOW_TO_BUILD.txt` را بخوانید یا `BUILD_APK.bat` را اجرا کنید.

**نکته:** APK برای Android 11+ (API 30) ساخته می‌شود.

## مجوزهای مورد نیاز

برنامه به مجوزهای زیر نیاز دارد:

- `READ_PHONE_STATE`: برای تشخیص وضعیت تماس
- `READ_CALL_LOG`: برای دسترسی به لاگ تماس‌ها
- `SEND_SMS`: برای ارسال پیامک
- `READ_PHONE_NUMBERS`: برای خواندن شماره تماس‌گیرنده

## نحوه استفاده

1. برنامه را نصب کنید
2. مجوزهای لازم را اعطا کنید
3. متن پیامک مورد نظر خود را وارد کنید
4. دکمه "ذخیره تنظیمات" را بزنید
5. سرویس را با استفاده از سوییچ فعال کنید

از این به بعد، هر زمان که تماسی رد شود یا پاسخ داده نشود، به صورت خودکار پیامک ارسال می‌شود.

## عیب‌یابی

### مشکل: پیامک ارسال نمی‌شود

- بررسی کنید که مجوز `SEND_SMS` اعطا شده باشد
- مطمئن شوید که سرویس فعال است
- بررسی کنید که متن پیامک خالی نباشد

### مشکل: تماس‌ها تشخیص داده نمی‌شوند

- بررسی کنید که مجوز `READ_PHONE_STATE` اعطا شده باشد
- برنامه را مجدداً راه‌اندازی کنید
- بررسی کنید که برنامه در پس‌زمینه در حال اجرا است
